==> GIT BASH COMMANDS <==
**************************

* ls -> list content of current location
Command: ls (enter)
* To open windows explorer
Command: start . (enter)
To open mac finder
Command: open . (enter)
* pwd (print working directory)
Command: pwd (enter)
* cd (change directory) & cd .. (to exit or navigate back from that location)
Command: cd (enter) Command: cd .. (enter)
* To Create File
Command: touch file_name.extension (enter)
Can make multiple files
Command: touch file1.extension file2.extension file3.extension (enter)
Can make file in specific folder or to the path
Command: touch path/fileName.extension (enter)
* To make folder
Command: mkdir folderName (enter)
* To remove the file
Command: rm FileName.extension (enter)
In order to delete directory or folder need to put or use the flag -rm
Command: rm -rf folderName (enter)
